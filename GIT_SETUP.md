# Git setup

## Initialize the repository

```bash
git init
git branch -M main
git add .
git commit -m "Initial commit"
```

## Add a remote

```bash
git remote add origin https://github.com/<your-user>/ioBroker.basic-ai.git
git push -u origin main
```

## Before publishing

Update these files with your real repository metadata:

- `package.json`
- `io-package.json`
- `README.md`
